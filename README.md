# Ambalan Sukanda Laksma — Website Resmi

Website resmi Ambalan Sultan Iskandar Muda & Laksamana Malahayati  
Pramuka SMAN 1 Sumberejo, Tanggamus, Lampung.

## 🏕️ Tentang

Ambalan Sukanda Laksma berdiri pada **27 Agustus 2006** dan merupakan satuan pramuka penegak yang berpangkalan di SMAN 1 Sumberejo. Nama ambalan diambil dari dua pahlawan besar Kesultanan Aceh — Sultan Iskandar Muda dan Laksamana Malahayati.

## 📁 Struktur Proyek

```
ambalan-sukanda-laksma/
│
├── index.html          # Halaman utama
├── about.html          # Tentang Ambalan (Sejarah, Sandi, Lambang, Pradana)
├── kegiatan.html       # Program Kegiatan
├── galeri.html         # Galeri Foto
├── angkatan.html       # Data Angkatan (Coming Soon)
├── struktur.html       # Struktur Organisasi (Coming Soon)
├── kontak.html         # Kontak & Sosial Media
│
├── assets/
│   ├── css/
│   │   └── style.css   # Stylesheet utama
│   ├── js/
│   │   └── script.js   # JavaScript utama
│   └── images/         # Folder gambar (tambahkan foto di sini)
│
└── components/
    ├── navbar.html     # Template navbar
    └── footer.html     # Template footer
```

## 🎨 Teknologi

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, Animations
- **Vanilla JavaScript** — Scroll effects, animations, interactions
- **Fonts** — Red Hat Display + Geist Mono (Google Fonts)
- **No frameworks** — Pure HTML/CSS/JS, zero dependencies

## 🚀 Cara Menjalankan

1. Buka file `index.html` di browser
2. Atau gunakan Live Server di VS Code
3. Atau deploy ke hosting statik (Netlify, GitHub Pages, dll.)

## 📸 Menambahkan Foto

Letakkan foto kegiatan di folder `assets/images/kegiatan/` dan ganti placeholder di `galeri.html` dengan tag `<img>` yang sesuai.
